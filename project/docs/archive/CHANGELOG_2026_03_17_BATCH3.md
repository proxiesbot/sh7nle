## ما تم تحسينه في هذه الدفعة

- توسيع صفحة **الجرد والتقارير** لتشمل:
  - صافي الربح بعد عمولات الإحالة
  - نسبة الهامش
  - عدد التجار
  - تفصيل الإيداعات اليدوية والأوتوماتيكية
  - أفضل المحيلين
  - جرد طرق الدفع
  - أحدث البانرات
- توسيع لوحة **المستخدمين** لتعرض:
  - نسبة استهلاك الرصيد
  - أرباح الإحالة الحالية والإجمالية
  - عدد المحالين
- توسيع صفحة **إدارة المستخدم** لتعرض:
  - رمز الإحالة
  - المُحيل
  - نسبة استهلاك الرصيد
  - ربحك من المستخدم
  - عدد المحالين
- تحسين الواجهة العامة:
  - ترويسة أفتح وأكثر وضوحًا
  - إبراز اسم **Sh7nle**
  - إظهار الأقسام مباشرة
  - إبراز السعر الخاص للتاجر على بطاقات المنتجات
- تحسين بيانات التجربة المحلية عبر seeders:
  - إضافة حساب تاجر تجريبي
  - إضافة بانر وإشعار عام تجريبيين
  - تعبئة بيانات ربح/تكلفة/IDs أوضح
