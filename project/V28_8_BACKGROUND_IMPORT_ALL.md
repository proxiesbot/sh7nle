# V28.8 Background Import All

تم إضافة استيراد شامل من داخل الموقع:
- زر "استيراد كل المنتجات" داخل مركز الاستيراد.
- يعمل تدريجيًا بالخلفية عبر polling.
- يعرض تقدم المزودات والمنتجات والحالي والفشل والمضاف والمحدث.
- يستورد كل المزودات الفعالة التي تدعم catalog.
- لا يكرر المنتجات الموجودة؛ يحدثها عند الحاجة.
- يمكن اختيار القسم الهدف قبل بدء الاستيراد، أو تركه تلقائيًا حسب أقسام المزود.
