# V28.57 Manual Fixes

تم تطبيق التعديلات المطلوبة يدويًا بدون Kiro:

## 1) زر تسجيل الخروج
- تمت إضافة زر خروج واضح في أعلى هيدر لوحة الإدارة حتى يظهر على الموبايل بدون النزول لآخر السايدبار.
- تم نقل زر الخروج داخل قائمة المستخدم العامة إلى أعلى القائمة، فوق روابط السوشال، حتى لا يحتاج المستخدم للنزول لأسفل القائمة.
- تم جعل أسفل سايدبار الإدارة ثابتًا/أخف حتى لا يختفي زر الخروج على الشاشات الصغيرة.

## 2) استيراد SW Games
- تم إلغاء التجميع التخمينـي لـ SW Games مثل: ألعاب / دردشة / بطاقات / اتصالات.
- صار SW Games يعرض الأقسام حسب مسار/مكان المنتج الحقيقي القادم من المزود، وليس حسب اسم المنتج.
- عند الدخول إلى مكان المنتج القادم من المزود تظهر المنتجات التابعة لهذا المكان فقط، وهذا يقلل الاستيراد العشوائي ويحافظ على مسار المزود.
- تم تحسين التقاط اسم وقسم SW Games من عدة حقول محتملة مثل gameName, game, serviceName, categoryName, parentName.

## 3) تبسيط لوحة الاستيراد
- حذف قسم “استيراد شامل بالخلفية” من الواجهة لأنه كان يزيد التخبيص.
- حذف لوحة الفحص الطويلة والتفاصيل غير الضرورية من الواجهة.
- تم تبسيط النصوص والأزرار بحيث يبقى المسار الأساسي: اختيار مزود، اختيار قسم محلي، تصفح أقسام المزود، استيراد منتج أو القسم الحالي.

## 4) تحسين تسجيل الدخول
- تم تخفيف البيانات المشتركة التي تُحمّل على صفحات تسجيل الدخول/التسجيل.
- صفحات Auth لم تعد تحمل أقسام المتجر، البنرات، النصوص العامة، وإعدادات الموقع الثقيلة.
- هذا يقلل زمن فتح صفحة الدخول وإكمال عملية الدخول.

## 5) Binance داخل طريقة الدفع
- عند اختيار Binance Pay تظهر إعدادات Binance مباشرة ضمن نفس الفورم.
- تمت إضافة حقول: Merchant ID, API Key, Secret Key, Webhook Secret, Currency, Return URL.
- Binance صار يُحفظ كطريقة دفع أوتوماتيكية مع تخزين إعداداته داخل config الخاص بطريقة الدفع.
- لا توجد أسرار مضافة في الكود؛ القيم يدخلها الأدمن من لوحة الإدارة.

- npm install: ناجح داخل بيئة الاختبار.
- npm run build: ناجح.
