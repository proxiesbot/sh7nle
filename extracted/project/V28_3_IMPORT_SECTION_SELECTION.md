# V28.3 Import target section selection

تمت إضافة اختيار واضح للقسم الهدف داخل مركز الاستيراد:
- اختر القسم الهدف للاستيراد من القائمة.
- إذا اخترت قسمًا، أي منتج أو قسم كامل تستورده من Shams أو SW Games سينزل مباشرة في هذا القسم.
- إذا تركته "تلقائي حسب أقسام المزود"، سيبقى السلوك السابق: إنشاء الأقسام حسب المزود.
- يوجد خيار منفصل للاستيراد إلى الصفحة الرئيسية.
