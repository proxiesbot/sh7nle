# V28.59 Local Section Destination Picker

- Replaced the flat "target section" dropdown in the provider import center with a cascading local destination picker.
- The first dropdown now shows main/root sections only.
- After choosing a main section, direct child sections appear in a second dropdown.
- If a child section has children, another level appears automatically.
- Leaving a child dropdown empty imports directly into the currently selected parent section.
- Admin can still choose automatic provider path mapping or direct import to the store homepage.
- The selected destination label is shown clearly before importing selected products.
