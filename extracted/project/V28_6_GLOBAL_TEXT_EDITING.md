# V28.6 Global Text Editing

تمت إضافة وضع تعديل نصوص عام يعمل في كل صفحات الموقع والإدارة:
- يظهر زر عائم للسوبر أدمن فقط: "تفعيل تعديل النصوص".
- عند تفعيله، اضغط على أي نص ظاهر في الصفحة ليظهر مربع تعديل.
- يتم حفظ النص في جدول content_texts.
- عند تحميل الصفحات، يتم استبدال النصوص المحفوظة تلقائيًا.
- هذا لا يلغي EditableText القديم، بل يكمله ويغطي المناطق التي لا تحتوي EditableText.
