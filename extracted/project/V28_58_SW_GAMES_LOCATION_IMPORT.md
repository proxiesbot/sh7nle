# V28.58 — SW Games Location-Based Import

- تم تعديل استيراد SW Games ليعتمد على مسار/مكان المنتج القادم من المزود نفسه، وليس على اسم المنتج أو كلمات مفتاحية.
- إذا أرسل المزود مسارًا مثل: `التطبيقات / Yoho Waka` سيتم إنشاء نفس المسار محليًا عند الاستيراد التلقائي.
- إذا كان المزود لا يرسل إلا مستوى واحد مثل `gameName`، سيتم استخدام هذا المستوى فقط لأن الـ API لا يوفّر أكثر منه.
- تم الحفاظ على ترتيب الأقسام حسب ظهورها في بيانات المزود قدر الإمكان بدل الفرز الأبجدي.
- تم تحديث نصوص لوحة الاستيراد لتوضح أن SW Games يعتمد على مكان المزود لا على الاسم.
